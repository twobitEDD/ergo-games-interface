# Web Workspace

Current scope:
- API routes for gameplay, wallet binding, on-chain intent lifecycle, and settlement scaffolding.
- UI scaffolding for mode trust messaging and deterministic game board placeholders.

Current scripts:
- `npm run test --workspace @ergo-games/web`: runs endpoint and server behavior tests.
- `npm run dev --workspace @ergo-games/web`: starts the local Next.js runtime using `src/app`.
- `npm run build --workspace @ergo-games/web`: compiles the web runtime for production.
